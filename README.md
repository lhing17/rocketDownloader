### 🏠 [项目主页](https://github.com/lhing17/rocketDownloader)

## 开发环境

- java(jdk) &gt;=11.0.0
- kotlin &gt;=1.3.0

## 项目构建

```sh
mvn clean package
```

## 执行单元测试

